# TeleSync Pro

电报群组消息克隆与自定义话术互动系统。

## 产品简介

TeleSync Pro 是一套面向 Telegram 群组场景的消息同步与互动工具，支持将源群组消息同步到目标群组，并提供多账号协同、自定义话术、多媒体发送等能力，用于构建更丰富的群组互动体验。

## 本次更新亮点

本次更新已支持以下内容类型：

- 图片
- 视频
- 文件
- GIF 动图
- Emoji 表情
- Sticker 贴纸
- 多媒体混合发送

现在的话术文件不再局限于纯文本，已可模拟更真实的群组聊天环境。

## 新增功能

- 指定账号发言
- 多账号互相回复
- 随机时间间隔发送
- 支持群话题（Topics）聊天
- 消息巡回循环
- 正在输入状态模拟
- 多媒体消息同步发送

这些功能可实现更接近真人的 Telegram 群组互动效果，让整体聊天氛围更自然。

## 核心优势

- **实时同步**：监控源群组消息并无缝同步至目标群组
- **智能模仿**：自动复制用户头像、名称，尽量保持原始对话风格
- **高度定制**：支持关键词替换、消息编辑、删除同步等功能
- **账号管理**：内置智能账号池，自动分配模仿账号并在超时后释放
- **界面简洁**：现代化 UI 设计，操作直观

## 应用场景

- **多语言群组桥接**：连接不同语言社区，提升交流效率
- **社区内容分流**：将主群消息同步到子群组，减轻管理压力
- **信息实时备份**：同步重要群组消息，降低信息丢失风险
- **内容分发**：将指定内容同步到多个目标群组

## 专业功能

- 支持消息转发、编辑、删除的完整同步
- 智能账号池管理，自动检测可用账号
- 关键词替换功能，便于处理敏感信息
- 用户排除系统，可自定义忽略特定用户
- 超时自动释放模仿账号，提高资源利用率
