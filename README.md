## 💡 产品简介

**TeleSync Pro** 是一套聚焦于 Telegram 群组管理、引流与互动场景的智能消息同步系统。它支持将源群组的消息实时克隆同步到目标群组，并结合多账号协同、自定义炒群话术、多媒体发送、关键词自动替换等能力，帮助运营者快速构建高活跃、更自然的群组互动体验。

## ✨ 核心亮点

- **⚡ 消息实时同步**：毫秒级监控源群组消息，快速克隆同步至目标群组。
- **🤖 多账号智能协同**：支持 TG 账号池管理、自动轮询分配、超时自动释放。
- **👥 真实炒群互动模拟**：支持指定账号发言、多账号互相回复、模拟“正在输入中”状态。
- **🖼️ 全媒体话术支持**：完美支持图片、视频、文件、GIF 动图、Emoji 表情、Sticker 贴纸混合发送。
- **⚙️ 灵活定制规则**：支持敏感关键词替换、指定用户排除、消息编辑与删除同步。
- **💬 适配 Topics 话题**：全面兼容 Telegram Group Topics 论坛模式话题聊天，使用更灵活。

## 🛠️ 功能特性

### 1. 自动互动与炒群能力
- 支持指定特定账号按照剧本发言
- 多账号矩阵互相回复，打造热烈讨论氛围
- 随机时间间隔发送，规避风控，更像真人
- 话术消息巡回循环，持续维持群内热度
- 发送前模拟“正在输入...”状态，真实度拉满

### 2. 同步与群管理能力
- 同步对方群组用户头像，简介，消息，等数据
- 1:1对方群组，固定模仿某位用户
- 支持源群消息的转发、编辑、删除等状态的完整同步
- 智能账号池管理，自动检测并剔除失效账号
- 强大的关键词替换功能，便于过滤敏感广告或替换为你自己的引流链接
- 用户排除系统，可自定义忽略源群特定用户的发言
- # 可用于克隆公群（简直神器）克隆搜索群，社群，等多种类型群组

## 🎯 应用场景

- **私域氛围构建**：通过多账号话术协同，快速提升新群组的活跃度与信任感。
- **社区内容分流**：将主群/官方频道消息实时同步到多个子群组，减轻多群管理压力。
- **多语言群组桥接**：配合业务需要连接不同语言社区，提升跨群交流效率。
- **信息实时备份**：同步重要加密群组消息到私密频道，降低被封群导致的信息丢失风险。
